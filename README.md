# MAC0332 Code Review

Pequena coleção de exercícios para a disciplina MAC0332.

## Scripts
- `solucoes.py`: contém as funções `sao_anagramas`, `cifra_de_cesar` e `valida_cpf`, todas aguardando implementação.

## Executando testes
No momento o repositório não possui suíte de testes automatizada. Sugestões de testes manuais:
- Confira se `sao_anagramas` identifica palavras com mesmas letras em diferentes ordens.
- Valide a cifra de César com deslocamentos positivos e negativos.
- Teste CPFs válidos e inválidos.

## Ambiente
Recomenda-se utilizar um ambiente virtual Python (`python -m venv .venv`) e instalar dependências adicionais conforme necessário.
